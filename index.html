My Projects

Discord Bot for UAlberta Undergraduate AI Society

The UAIS student club had a problem — manual discord server management is boring. Every time an event came up, an executive had to notify everyone about it in the server. My friend reached out to me, as I have had previous experience developing Discord bots, so I took charge of API integration, feature development, repository cleanup and maintenance, and overall project organization.

My most important contribution to the project has been the Google Calendar integration — convenient automatic reminders about upcoming events, and an interactive UI within Discord to create events. Additionally, I helped other team members with discord.py — reviewed their code, and made sure everything worked together seamlessly.
